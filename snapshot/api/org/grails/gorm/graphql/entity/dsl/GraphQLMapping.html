Package: org.grails.gorm.graphql.entity.dsl

[Groovy] Class GraphQLMapping

    • org.grails.gorm.graphql.entity.dsl.GraphQLMapping

    • Builder to provide GraphQL specific data for a GORM entity Usage: 

       static graphql = {
     exclude 'foo'
     add new AdditionalGraphQLProperty().name('bar').type(String)
     description 'Business users'
 
 }
      The closure can also be passed to GraphQLMapping.build to provide code completion

      James Kleeh
